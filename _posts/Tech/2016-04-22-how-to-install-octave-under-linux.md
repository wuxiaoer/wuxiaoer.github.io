## octave安装流程

1. 下载地址
[octave down](http://www.gnu.org/software/octave/)

2. 安装
   首先解压:

   ```
   fatansy@fantasy:sw$ tar zxvf octave-4.0.2.tar.gz
   ```

   然后进入源代码目录下:

   ```
   fatansy@fantasy:sw$ cd octave-4.0.2/
   ```

   配置:

   ````
   ./configure
   ```

   编译安装:

   ```
   make
   sudo make install
   ```
